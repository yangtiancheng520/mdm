# 字段分类管理使用指南

## 菜单位置

**数据标准与模型中心 → 字段分类管理**

## 完整菜单结构

```
📁 数据标准与模型中心
├─ 📂 字段分类管理 ← 在这里维护
├─ 📄 字段标准库
├─ 📄 数据标准视图
├─ 📄 编码规则
├─ 📄 值域管理
└─ 📄 变更审批配置
```

## 使用步骤

### 1. 登录系统
- 访问：http://localhost:9000/
- 账号：admin
- 密码：123456

### 2. 刷新菜单
- 如果看不到"字段分类管理"菜单，请刷新浏览器页面（F5）

### 3. 进入分类管理
- 点击左侧菜单：数据标准与模型中心
- 展开后点击：字段分类管理

### 4. 功能操作

**创建分类：**
- 点击右上角"+ 新增顶级分类"按钮
- 填写分类编码、名称、排序等信息
- 点击"确定"保存

**创建子分类：**
- 点击分类节点上的"新增"按钮
- 自动关联父级分类

**编辑分类：**
- 点击分类节点上的"编辑"按钮
- 修改分类信息

**删除分类：**
- 点击分类节点上的"删除"按钮
- 注意：必须没有子分类和关联字段才能删除

## 默认分类数据

系统已预置以下分类：
```
📁 基础信息 (BASIC)
   ├─ 标识类 (IDENTITY)
   └─ 名称类 (NAME)
📁 联系信息 (CONTACT)
📁 财务信息 (FINANCIAL)
📁 管理属性 (ADMIN)
```

## 在字段标准库中使用

1. 进入"字段标准库"页面
2. 点击"新增"或"编辑"字段
3. 在"字段分类"下拉框中选择分类
4. 保存字段

## 筛选字段

在字段标准库页面：
- 使用搜索栏的"分类"下拉框筛选
- 选择分类后点击"查询"

## 常见问题

**Q: 看不到"字段分类管理"菜单？**
A: 刷新浏览器页面或重新登录

**Q: 删除分类失败？**
A: 检查是否还有子分类或关联字段

**Q: 分类下拉框为空？**
A: 需要先在"字段分类管理"中创建分类
