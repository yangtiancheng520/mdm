# MDM 管理系统 - 启动说明

## 🚀 快速启动

### 一键启动所有服务
双击运行 `start-all.bat`

### 单独启动服务
- **启动后端**: 双击 `start-backend.bat`
- **启动前端**: 双击 `start-frontend.bat`

### 停止服务
双击运行 `stop-all.bat`

---

## 📋 手动启动命令

### 后端 (Spring Boot)
```bash
cd backend
mvn spring-boot:run
```

### 前端 (Vue 3)
```bash
cd frontend
npm run dev
```

## ⚠️ 注意事项

1. 首次运行前请确保已导入数据库：
   ```bash
   # 1. 创建数据库和表结构
   mysql -uroot -padmin < db/mdm_schema.sql
   
   # 2. 导入初始化数据
   mysql -uroot -padmin < db/mdm_data.sql
   ```

2. 如果端口被占用，可以修改配置：
   - 后端端口: `backend/src/main/resources/application-dev.yml`
   - 前端端口: `frontend/vite.config.ts`

3. Maven 环境配置：
   - MAVEN_HOME = D:\maven\mvn
   - PATH 已添加 D:\maven\mvn\bin

4. Redis 服务需要启动：
   - 默认端口: 6379
   - 配置文件: `application-dev.yml`
