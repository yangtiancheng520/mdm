# MDM 主数据管理平台 - 启动说明

## 服务状态

### ✅ 前端服务
- **地址：** http://localhost:9000/
- **状态：** 已启动
- **启动脚本：** `start-frontend.bat`

### ⚠️ 后端服务
- **地址：** http://localhost:3000/
- **状态：** 需要手动启动
- **启动脚本：** `start-backend.bat`

## 启动步骤

### 方式一：使用启动脚本（推荐）

1. 双击运行 `start-backend.bat`
2. 等待出现 "Started MdmBackendApplication" 字样
3. 浏览器访问 http://localhost:9000/

### 方式二：使用IDE

1. 打开 IntelliJ IDEA 或 Eclipse
2. 导入项目 `backend`
3. 找到主类 `MdmBackendApplication.java`
4. 右键 → Run 'MdmBackendApplication'

### 方式三：使用命令行

```bash
cd backend
mvn spring-boot:run
```

## 验证服务

### 后端服务检查
- 访问：http://localhost:3000/swagger-ui.html
- 或执行：`curl http://localhost:3000/api/health`

### 前端服务检查
- 访问：http://localhost:9000/
- 应显示登录页面

## 测试字段分类管理功能

1. 登录系统（账号：admin，密码：admin123）
2. 刷新页面，确保菜单加载
3. 导航到：**数据标准与模型中心 → 字段分类管理**
4. 测试功能：
   - 创建分类
   - 编辑分类
   - 删除分类（需无子分类和关联字段）
5. 导航到：**数据标准与模型中心 → 字段标准库**
6. 测试功能：
   - 按分类筛选字段
   - 新增字段时选择分类
   - 编辑字段时修改分类

## 常见问题

### Q: 前端502错误？
**A:** 后端未启动，请先启动后端服务

### Q: 菜单不显示？
**A:** 刷新页面或重新登录

### Q: Redis连接失败？
**A:** 不影响功能，系统会自动降级处理

## 技术支持

如遇问题，请检查：
1. 后端日志：`backend/logs/mdm-backend.log`
2. 浏览器控制台：F12 → Console
3. 数据库连接：确保 MySQL 服务已启动
