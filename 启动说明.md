# MDM 管理系统 - 启动说明

## 🚀 快速启动

### 一键启动所有服务
双击运行 `start-all.bat`

### 单独启动服务
- **启动后端**: 双击 `start-backend.bat`
- **启动前端**: 双击 `start-frontend.bat`

### 停止服务
双击运行 `stop-all.bat`

---

## 📋 手动启动命令

### 后端 (Spring Boot)
```bash
cd backend
mvn spring-boot:run
# 或使用 mvnd（更快）
mvnd spring-boot:run
```

### 前端 (Vue 3)
```bash
cd frontend
npm run dev
```

## 📁 项目结构

```
mdm/
├── backend/          # Spring Boot 后端
│   ├── src/
│   └── pom.xml
├── frontend/         # Vue 3 前端
│   ├── src/
│   └── package.json
├── db/              # 数据库脚本
│   └── vue_admin.sql
├── start-all.bat    # 一键启动
├── start-backend.bat
├── start-frontend.bat
└── stop-all.bat     # 停止服务
```

---

## ⚠️ 注意事项

1. 首次运行前请确保已导入数据库：
   ```bash
   mysql -uroot -padmin < db/vue_admin.sql
   ```

2. 如果端口被占用，可以修改配置：
   - 后端端口: `backend/src/main/resources/application.yml`
   - 前端端口: `frontend/vite.config.ts`

3. Maven 环境变量已配置：
   - MAVEN_HOME = D:\maven
   - PATH 已添加 D:\maven\bin
