# 值域管理字段映射说明

## 数据库表结构（V2.0 创建）

```sql
CREATE TABLE std_value_domain_item (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    domain_id BIGINT NOT NULL,
    item_code VARCHAR(100) NOT NULL,      -- 项编码
    item_name VARCHAR(200) NOT NULL,      -- 项名称
    parent_item_id BIGINT,
    sort INT DEFAULT 0,
    status VARCHAR(20) DEFAULT 'active',
    ...
);
```

## 字段含义

- **item_code**: 编码（唯一标识）
  - 示例：`LIMITED_COMPANY`, `JOINT_STOCK`
  - 用途：系统内部使用，作为唯一标识

- **item_name**: 项名称（显示文本）
  - 示例：`有限责任公司`, `股份有限公司`
  - 用途：显示给用户看的文本

## 数据流

### 前端 → 后端（保存）

```
前端输入：
- 编码输入框：`item_code` (如: 'LIMITED_COMPANY')
- 项值输入框：`item_name` (如: '有限责任公司')

后端保存：
- item_code = opt.code
- item_name = opt.value
```

### 后端 → 前端（读取）

```
数据库：
- item_code = 'LIMITED_COMPANY'
- item_name = '有限责任公司'

返回给前端：
- code = item_code = 'LIMITED_COMPANY'
- value = item_name = '有限责任公司'
- label = item_name = '有限责任公司'
```

## 前端显示

| 编码 | 项值 |
|------|------|
| LIMITED_COMPANY | 有限责任公司 |
| JOINT_STOCK | 股份有限公司 |
| PARTNERSHIP | 合伙企业 |

## 实体类映射

```java
@Column(name = "item_code", length = 100)
private String itemCode;

@Column(name = "item_name", length = 200)
private String itemValue;  // 注意：Java字段名是itemValue，但映射到item_name字段
```

## 注意事项

1. 实体类中的 `itemValue` 字段映射到数据库的 `item_name` 字段
2. 前端的 `value` 对应数据库的 `item_name`
3. 前端的 `code` 对应数据库的 `item_code`
