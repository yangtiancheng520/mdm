### 3. 按钮权限使用示例

在前端组件中检查权限：

```typescript
import { hasPermission } from '@/utils/permission'

// 检查是否有新增规则权限
if (hasPermission('quality:rule:create')) {
  // 显示新增按钮
}

// 检查是否有测试规则权限
if (hasPermission('quality:rule:test')) {
  // 显示测试按钮
}
```
