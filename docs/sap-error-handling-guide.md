# SAP分发错误处理和优化

## 📋 常见SAP错误及解决方案

### 1️⃣ 计量单位相关错误

#### 错误1：计量单位KG用作代替单位
```
错误信息：计量单位KG用作代替单位 [KG]
原因：KG被SAP配置为代替单位，不能作为基本单位
解决方案：
- 使用其他单位：PC、EA、SET、L
- 或联系SAP顾问修改单位配置
```

#### 错误2：基准计量单位不能更改
```
错误信息：基准计量单位不能更改；选择"显示错误"
原因：物料已存在，且基本单位与传入的不同
解决方案：
- 系统已优化：自动先尝试UPDATE，失败再CREATE
- 确保传入的基本单位与SAP中已有物料一致
```

## ✅ 分发逻辑优化

### 原逻辑
```java
// 根据本系统的分发历史判断
if (上次分发成功) {
    执行UPDATE
} else {
    执行CREATE
}
```

**问题：** 物料可能通过其他方式已在SAP中存在

### 新逻辑
```java
// 先尝试更新，失败再创建
result = connector.update(materialCode, data);
if (!result.success) {
    result = connector.create(data);
}
```

**优势：**
- ✅ 自动处理已存在的物料
- ✅ 无需依赖本系统历史记录
- ✅ 兼容通过其他途径创建的物料

## 🔧 字段状态判断优化

### 智能错误匹配
```java
// 判断字段失败状态
1. 字段名匹配：检查错误消息中的字段名
2. 字段中文名匹配：检查错误消息中的中文名称
3. 特殊字段识别：
   - 计量单位字段 + "计量单位"关键词
   - 物料组字段 + "物料组"关键词
```

### 示例
```
错误：基准计量单位不能更改

匹配结果：
- MEINS (基本计量单位) → FAILED ✓
- BASE_UOM → FAILED ✓
- 其他字段 → SUCCESS ✓
```

## 📊 SAP物料单位最佳实践

### 推荐的基本单位
```
✅ 可用作基本单位：
- PC (件/Piece)
- EA (个/Each)
- SET (套/Set)
- L (升/Liter)
- G (克/Gram)
- ST (件/Stück)

⚠️ 可能是代替单位：
- KG (千克/Kilogram)
- M (米/Meter)
```

### 单位换算配置
```sql
-- 建立单位换算关系
示例：饮料产品
基本单位：瓶 (Bottle)
代替单位：
- 箱：1箱 = 12瓶
- KG：1KG = 2瓶 (根据产品密度)
- L：1L = 4瓶 (250ml/瓶)
```

## 🎯 数据维护建议

### 1. 新建物料
```
步骤：
1. 在数据维护页面创建物料
2. 选择合适的基本单位（PC、EA、SET等）
3. 填写必填字段
4. 分发到SAP
```

### 2. 更新物料
```
步骤：
1. 系统自动识别物料是否存在
2. 自动使用UPDATE模式
3. 注意：基本单位不能更改
```

### 3. 批量分发
```
建议：
- 分批执行，每批50-100条
- 检查失败的记录
- 根据错误信息调整数据
```

## 📝 错误处理流程

```
分发失败
   ↓
查看错误信息
   ↓
├─ 计量单位错误
│    ↓
│  更改基本单位 → PC/EA/SET
│
├─ 物料已存在
│    ↓
│  系统自动UPDATE（已优化）
│
└─ 其他错误
     ↓
   查看SAP消息 → 联系SAP顾问
```
