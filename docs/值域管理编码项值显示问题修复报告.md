# 值域管理编码项值显示问题修复报告（最终版）

## 问题描述

在值域管理界面，维护值域项时出现"编码"和"项值"显示相同的问题：
- **错误显示**：编码显示 "01"，项值也显示 "01"
- **期望显示**：编码显示 "01"，项值显示 "钢材类"

## 问题根源分析

### 1. 数据库表结构

根据迁移脚本，`std_value_domain_item` 表的实际字段：

**V7.0 创建表**：
```sql
CREATE TABLE std_value_domain_item (
    id BIGINT PRIMARY KEY,
    domain_id BIGINT NOT NULL,
    item_value VARCHAR(200),    -- 值
    item_label VARCHAR(200),    -- 显示名
    ...
);
```

**V18.0 添加字段**：
```sql
ALTER TABLE std_value_domain_item
ADD COLUMN item_code VARCHAR(100);  -- 编码
```

**最终字段结构**：
- `item_code`：编码（唯一标识）
- `item_value`：值（存储值）
- `item_label`：显示名（显示给用户的文本）

### 2. 实体类字段缺失

实体类 `ValueDomainItem.java` 原本只有：
- `itemCode` → `item_code`
- `itemValue` → `item_value`

**缺少 `itemLabel` 字段**，导致无法读取显示名。

### 3. 数据流设计

**前端设计**：
- 编码输入框：`opt.code`
- 项值输入框：`opt.value`

**后端应该保存**：
- `item_code` = `opt.code`（编码）
- `item_value` = `opt.code`（存储值，与编码一致）
- `item_label` = `opt.value`（显示名，用户输入的项值）

**后端应该返回**：
- `code` = `item_code`（编码）
- `value` = `item_label`（项值，显示名）
- `label` = `item_label`（显示名）

## 修复方案

### 1. 添加 itemLabel 字段到实体类 ✅

**文件**：`backend/src/main/java/com/vueadmin/entity/standard/ValueDomainItem.java`

```java
/**
 * 显示名（显示给用户的标签）
 */
@Column(name = "item_label", length = 200)
private String itemLabel;
```

### 2. 修正保存逻辑 ✅

**文件**：`backend/src/main/java/com/vueadmin/service/standard/ValueDomainService.java`

```java
private void saveDomainItems(Long domainId, List<DomainOptionDto> options) {
    for (DomainOptionDto opt : options) {
        if (opt.getCode() != null && !opt.getCode().trim().isEmpty()) {
            ValueDomainItem item = new ValueDomainItem();
            item.setDomainId(domainId);
            item.setItemCode(opt.getCode());        // 编码
            item.setItemValue(opt.getCode());       // 值 = 编码（用于存储）
            item.setItemLabel(opt.getValue());      // 显示名 = 项值
            item.setSort(opt.getSort() != null ? opt.getSort() : 0);
            item.setStatus("启用");
            valueDomainItemRepository.save(item);
        }
    }
}
```

### 3. 修正读取逻辑 ✅

**文件**：`backend/src/main/java/com/vueadmin/service/standard/ValueDomainService.java`

```java
public List<DomainOptionDto> getOptionsByDomainId(Long domainId) {
    List<ValueDomainItem> items = valueDomainItemRepository
            .findByDomainIdAndStatusOrderBySortAsc(domainId, "启用");
    return items.stream()
            .map(item -> new DomainOptionDto(
                item.getItemCode(),                                                      // code: 编码
                item.getItemLabel() != null ? item.getItemLabel() : item.getItemValue(), // value: 显示值
                item.getItemLabel() != null ? item.getItemLabel() : item.getItemValue(), // label: 显示名
                item.getSort()
            ))
            .collect(Collectors.toList());
}
```
